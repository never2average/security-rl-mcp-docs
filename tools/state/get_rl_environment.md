# `get_rl_environment`

Fetch current state, missing sections, validation errors, and agent run history.

| Field | Value |
| --- | --- |
| MCP page name | `get_rl_environment` |
| Current implementation | `get_rl_environment` |
| Auth | Google/Dex OIDC bearer token, or temporary service bearer token |

Use this tool from a connected coding agent through `https://security-rl.useimmaculate.com/mcp`.
