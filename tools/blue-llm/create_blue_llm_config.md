# `create_blue_llm_config`

Register a base model, system prompt, and metadata for a Blue-team LLM candidate.

| Field | Value |
| --- | --- |
| MCP page name | `create_blue_llm_config` |
| Current implementation | `create_blue_llm_config` |
| Auth | Google/Dex OIDC bearer token, or temporary service bearer token |

Use this tool from a connected coding agent through `https://security-rl.useimmaculate.com/mcp`.
