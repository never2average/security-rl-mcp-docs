# `log_red_team_issue`

Create a versioned Red finding. Blue must fix it by `flaggedAtVersion + 1`.

| Field | Value |
| --- | --- |
| MCP page name | `log_red_team_issue` |
| Current implementation | `log_red_team_issue` |
| Auth | Google/Dex OIDC bearer token, or temporary service bearer token |

Use this tool from a connected coding agent through `https://security-rl.useimmaculate.com/mcp`.
