# `get_red_blue_agent_brief`

Return the role-specific brief for an agent session.

| Field | Value |
| --- | --- |
| MCP page name | `get_red_blue_agent_brief` |
| Current implementation | `get_red_blue_agent_brief` |
| Auth | Google/Dex OIDC bearer token, or temporary service bearer token |
